                                                      # 重要公式汇总

                                                      # 代数公式

                                                      • 二次方程求根公式：x = [-b ± √(b²-4ac)] / 2a
                                                      • 韦达定理：x₁+x₂ = -b/a, x₁·x₂ = c/a
                                                      • 二项式定理：(a+b)ⁿ = Σ(k=0 to n) C(n,k)·aⁿ⁻ᵏ·bᵏ

                                                      # 三角函数公式

                                                      • 基本关系：sin²α + cos²α = 1, tan α = sin α / cos α
                                                      • 和差公式：sin(α±β) = sin α cos β ± cos α sin β
                                                      • 二倍角公式：sin 2α = 2sin α cos α, cos 2α = cos²α - sin²α

                                                      # 导数公式

                                                      • (xⁿ)' = nxⁿ⁻¹
                                                      • (sin x)' = cos x, (cos x)' = -sin x
                                                      • (eˣ)' = eˣ, (ln x)' = 1/x

                                                      # 积分公式

                                                      • ∫xⁿ dx = xⁿ⁺¹/(n+1) + C (n≠-1)
                                                      • ∫sin x dx = -cos x + C, ∫cos x dx = sin x + C

                                                      # 向量公式

                                                      • 向量模长：|a| = √(x²+y²+z²)
                                                      • 向量数量积：a·b = |a||b|cos θ
                                                      • 向量叉积：|a×b| = |a||b|sin θ
