---


```text
Collection 接口（单列）
├─ List 接口（有序、可重复、有索引）
│    ├─ ArrayList
│    ├─ LinkedList
│    └─ Vector（已淘汰）
│
├─ Set 接口（无序、不可重复、无索引）
│    ├─ HashSet
│    ├─ LinkedHashSet
│    └─ TreeSet
│
└─ Queue 接口（队列，极少用）

Map 接口（双列，key-value，key不可重复）
├─ HashMap
├─ LinkedHashMap
├─ TreeMap
├─ Hashtable（已淘汰）
└─ Properties（常用）

```
