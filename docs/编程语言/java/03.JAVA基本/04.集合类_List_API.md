### List 实现类

#### ArrayList
```text
底层：动态数组，查询快、增删慢，线程不安全，开发最常用。
```
#### LinkedList
```text
底层：双向链表，增删快、查询慢，线程不安全，适合频繁插入删除、做队列 / 栈。
```
#### Vector (废弃)

#### ArrayDeque
```text
双端队列，可当 List / 栈 / 队列用，效率高于 Stack、LinkedList
```

#### CopyOnWriteArrayList
```text
JUC 并发包，线程安全，读多写少场景用
```
