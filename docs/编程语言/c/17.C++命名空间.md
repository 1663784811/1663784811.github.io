### 命名空间定义
```cpp
namespace 空间名{
    int a = 1;
    
    class MyClass {
        public:
            void sayHello();
    }
}
```
### 命名空间使用
```cpp
方式1：
空间名::a;

方式2
using namespace 空间名;


```
