## 标准
```text
TTL   电平   (  4根线  0-3.3V  )

RS232 标准   (  9根线  -15到15V )
RS422          9根线
RS485          9根线

```

## TTL接线:

```text
   设备1            设备2              
    1. 3v3             3v3
    2. GND             GND
    3. TX              RX
    4. RX              TX
```
