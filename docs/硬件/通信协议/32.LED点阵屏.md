LED 8*8 点阵屏模块是一种常见的 LED 屏幕模块，<br>
它由 8 行 8 列的 LED 点阵组成。
共有 64 个发光二极管。<br>   
每个 LED 点可以控制亮灭 <br>

直接使用 I/O 口驱动，占用较多的 I/O 口资源，特别是随着点阵屏的数量增加，<br>
所以一般的应用，会选择专用的驱动芯片，例如 74HC595，MAX7219 等等
