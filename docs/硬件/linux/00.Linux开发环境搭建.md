### 安装NFS 
```shell
# 服务端
sudo apt-get install nfs-kernel-server



```

````shell
# 配置NFS
sudo vi /etc/exports
# 添加nfs 目录到最后
/home/why/linux/nfs *(rw,sync,no_root_squash)
# 重启NFS
sudo /etc/init.d/nfs-kernel-server restart

````
